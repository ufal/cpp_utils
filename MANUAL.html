1. Introduction

UFAL C++ Utils is a small C++ cross-platform library used in several UFAL projects. It is released under MPL 2.0 license and is versioned using Semantic Versioning.

The supported platforms are Linux, OS X and Windows, supported compilers are gcc, clang, tdm-gcc on Windows and Visual C++ 2015 or later on Windows.

UFAL C++ Utils repository http://github.com/ufal/cpp_utils is hosted on GitHub.
